# Alpigo's Item Config System
You can get Growtopia's item.dat list with these app. You can use a API or our Discord bot.

## ⚠️ Warning
This app is not affiliated with Ubisoft Entertainment. Everyday our API/Bot refresh the data.
> You can use some commands for edit or get items.dat file.
 
> [!TIP]
> You can use our commands via our Discord bot.

```
/help
```
> List for commands.

```
/get <items.dat or items.html>
``` 
> Gives you a items.dat or items.html file.

```
/api <your api license key>
```
```
/edit <line: item.dat line> or <id: item id>
```
> You can edit items.dat

> Enter your license key for API key.
>
